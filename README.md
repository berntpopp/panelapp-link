# PanelApp-Link

Read-only MCP/API server that mirrors [PanelApp](https://panelapp.genomicsengland.co.uk/)
(Genomics England UK + PanelApp Australia) into local SQLite and answers
panel/gene questions over both regions.

A drop-in sibling of the `*-link` fleet (e.g. `gencc-link`, `hgnc-link`).

> documentation (architecture, usage, data lifecycle) lands in the W8 packaging
